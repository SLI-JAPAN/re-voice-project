# Re:Voice Project

話せるから、整う。整うから、進める。

## 概要

Re:Voiceは、適性診断と1on1で、評価ではなく「自分らしい力の出し方」を見つける4ヵ月間のプログラムです。

## プロジェクト内容

- 社員全員との1対1面談
- NET*ASK適性診断の実施
- 個別マネジメント戦略会議
- 総括報告と次のアクションの提案
